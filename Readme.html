简介

这是一款基于PHP+MYSQL的国产单用户博客程序，简单明快，高效，强大。

— 小牛哥(Dyhb)

程序功能:

  1. AJAX+嵌套评论+分页（评论音乐，评论微博客，评论相片，评论日志，单独留言板）
  2. 自定义伪静态样式，比如说：http://blog.56swun.com/mylife/school/i_eat_a_dog.html,本网址不存在（支持分类，日期，别名，前缀）
  3. 自定义衔接静态化，比如说：http://blog.56swun.com/news/2010-09-10/1214-3-1.html，静态化的日志存在
  4. 多级分类，告别单调的一级分类
  5. 语言包功能，一键切换语言（默认一个英文包，一个简体中文包）
  6. 前台快速发布，像论坛一样叼，cms，博客，论坛一体化
  7. 大附件批量上传，自带水印，相册
  8. 投稿注册，权限...
  9. 更多功能等待你的发掘...

安装

  1. 将 DYHB.blog 压缩包解压至一个空文件夹。
  2. 上传Upload中所有文件到服务器
  3. 上传后，请设置以下指定的文件或目录的权限为777：（非windows环境）
    需要立刻设置权限：
    includes/config.php文件权限为777(配置文件)
    width/cache文件夹权限为777(缓存文件夹)
    可以以后设置权限：
    width/upload(附件目录)
    width/upload/default(默认所有附件放在一个文件夹)
    width/backup(数据备份目录)
    images/qq(头像目录)
    images/mp3/mp3player.xml(音乐播放器xml文件)
  4. 在浏览器中访问install.php，DYHB.blog 会自动建立数据库表。程序会自动检测你是否连接上数据库，如果一直提示错误，请仔细检查你填写的数据库信息是否正确。要是依然错误，请前往DYHB-BLOG官方论坛寻求解决方法。
  5. 请安装完后，一定要登录后台更新一次全站缓存，否则程序不能运行约，小小提醒哈!
